About the app This is a very simple app intended to work with ParkingMeter REST API:    https://github.com/mwisnie/parkingMeter/tree/develop

Built with Angular 6 and Angular Materials, it allows to:

  • add, modify and delete parking spaces
  • assign cars to spaces (creating parking sessions associated with spaces)
  • modify parking sessions
  • calculate cost of a session based on it's parameters (session length and tariff)
  • calculate total income from all sessions

It's goal is to illustrate how the API actually works, rather than being actual application for managing parking.
